Webscraper

MAL Webscraper

Project description: This project is built with the BeautifulSoup module. This project scrapes the MyAnimeList(MAL) website, and grabs valuable information for each anime card. The user specifies the season and any other filters such as genre, score, anime format, studio, number of members, or source material. All anime fitting the user's criteria are then displayed in text files matching the page number they were found on, along with the release date of the anime, and a link to find additional information about the specific anime title.
