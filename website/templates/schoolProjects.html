Rent-a-bike

Twitterverse

Project description: This project was built in Python 3 and was unit tested. This project looks through a file of Twitter users and filters out certain ones based on specifications made.

Turtle Graphics-Pro

Project description: This project was built in C and used the linked list data structure. The function of this project is to take instructions for a virtual pen on which direction to move towards and what distance to move to draw onto a virtual paper.





Quadtree

Project description: This project was built in C and used the binary search tree data structure. This a tool which takes an input image and looks at the colours in the image to split it, by the number of specified splits, into rectangles where each rectangle is a uniform colour.
